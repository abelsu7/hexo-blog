1. os/exec

1.1 只执行命令，不要输出结果

执行命令可以使用Run()或者Start()方法，Run()是阻塞的执行，Start()是非阻塞的执行：

package main

import (
    "fmt"
    "os/exec"
)

func main() {
    command := exec.Command("ping","www.baidu.com")
    err := command.Run() // 阻塞执行
    if err != nil{
        fmt.Println(err.Error())
    }
}
