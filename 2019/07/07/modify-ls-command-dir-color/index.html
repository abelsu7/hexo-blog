Linux 终端修改 ls 命令目录显示颜色

2019-07-07
  • Linux

保护眼睛，猿猿有责

在.bashrc或.zshrc中添加：

export LS_COLORS=${LS_COLORS}:'di=01;37;44'

之后source ~/.bashrc，即可在当前终端生效。
