Linux下的压缩包，最常见的格式是.tar.gz以及.tar.bz2。而Linux平台最常用的压缩解压命令就是tar命令，相信不少人都有面对不同格式压缩包的各种参数抓狂的经历，今天就来总结学习一下tar命令的基本操作。

tar -zxvf ***.tar.gz
tar -xvf ***.gz
# tar.xz
xz -d ***.tar.xz
tar -xvf  ***.tar
# 或直接
tar -xvJf ***.tar.xz
